Project: Bitcoin.in.th Merchant Account
Module: Joomla: Virtuemart 2 Module Payment
Author: dave@advancedstyle.com

Installation:
1) Install Joomla and Virtuemart 2 (this module will not work with other versions of Virtuemart)
2) Create a merchant account at http://bitcoin.in.th/merchant-account/
3) Go to your Joomla Admin -> Extensions -> Extensions Manager -> Upload the Zip file package under the "Upload Package File" section
4) Go to Joomla Admin -> Components -> Virtuemart -> Shop -> Payment Methods
5) Click "New" -> Select "Bitcoin.in.th" from the Payment Method list
6) Click "Save"
7) Click the "Configuration" tab and enter your API ID and API Key from bitcoin.in.th
